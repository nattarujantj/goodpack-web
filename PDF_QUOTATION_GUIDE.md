# คู่มือการใช้งานฟีเจอร์ PDF ใบเสนอราคา

## ภาพรวม
ฟีเจอร์นี้ช่วยให้คุณสามารถสร้างและพิมพ์ใบเสนอราคาเป็นไฟล์ PDF ได้อย่างง่ายดาย

## ขั้นตอนการใช้งาน

### 1. การเข้าถึงฟีเจอร์
1. เปิดแอปพลิเคชัน
2. ไปที่หน้า "รายการเสนอราคา" (Quotation List)
3. เลือกใบเสนอราคาที่ต้องการพิมพ์
4. ไปที่หน้า "รายละเอียดเสนอราคา" (Quotation Detail)

### 2. การสร้างและพิมพ์ PDF
1. ในหน้า "รายละเอียดเสนอราคา" ให้เลื่อนลงไปด้านล่าง
2. คลิกปุ่ม **"พิมพ์ PDF ไทย"** (สีเขียว)
3. ระบบจะแสดง dialog "กำลังสร้าง PDF ไทย..."
4. เมื่อสร้างเสร็จแล้ว จะเปิด dialog สำหรับการพิมพ์/บันทึก

### 3. ตัวเลือกในการพิมพ์/บันทึก
- **พิมพ์**: เลือกเครื่องพิมพ์และพิมพ์ออกมา
- **บันทึกเป็น PDF**: บันทึกไฟล์ PDF ลงในอุปกรณ์
- **แชร์**: แชร์ไฟล์ PDF ผ่านแอปอื่นๆ

## การแก้ไขปัญหา

### ปัญหาที่อาจเกิดขึ้น
1. **PDF ไม่แสดงภาษาไทย**: ระบบจะใช้ฟอนต์ default หากไม่พบฟอนต์ไทย
2. **การสร้าง PDF ช้า**: ขึ้นอยู่กับจำนวนรายการสินค้าและข้อมูลลูกค้า
3. **ไม่สามารถพิมพ์ได้**: ตรวจสอบการเชื่อมต่อเครื่องพิมพ์

### การแก้ไข
- หากมีปัญหา ให้ลองปิดและเปิดแอปใหม่
- ตรวจสอบการเชื่อมต่ออินเทอร์เน็ต
- ตรวจสอบสิทธิ์การเข้าถึงเครื่องพิมพ์

## ข้อกำหนดระบบ
- Flutter 3.5.0 ขึ้นไป
- Package `pdf: ^3.10.7`
- Package `printing: ^5.12.0`
- Package `google_fonts: ^6.2.1`

## ฟีเจอร์ PDF ไทย

### PDF ไทย (Enhanced)
- Layout แบบมืออาชีพ เหมือนใบเสนอราคาจริง
- มีโลโก้บริษัท GPS
- ข้อมูลบริษัทครบถ้วน
- ตารางสินค้าแบบมาตรฐาน
- สรุปยอดรวมแบบละเอียด
- ช่องลายเซ็น
- รองรับฟอนต์ไทยได้ดีขึ้น
- ใช้ Google Fonts สำหรับฟอนต์ไทย
- แปลงยอดรวมเป็นตัวอักษรไทย
- วันที่แบบปฏิทินไทย

## การพัฒนาต่อ
หากต้องการปรับแต่ง PDF:
1. แก้ไขไฟล์ `lib/services/pdf_service_thai_enhanced.dart`
2. ปรับแต่ง layout, สี, หรือฟอนต์
3. เพิ่มข้อมูลเพิ่มเติมตามต้องการ

## หมายเหตุ
- PDF จะถูกสร้างในรูปแบบ A4
- ไฟล์ PDF จะมีชื่อเป็น `ใบเสนอราคา_[รหัสใบเสนอราคา].pdf`
- ระบบรองรับการพิมพ์และบันทึกไฟล์
